# 파수꾼의 취재 소스 (공개)

> 소스의 선택 자체가 편집 스탠스의 일부다. 그래서 이 목록도 공개한다. (강령 6)
> 운영하며 소스를 더하고 뺄 수 있으며, 변경은 git 히스토리에 남는다.

## 수집 원칙

1. **사실과 여론을 분리해 수집한다.** 사실은 복수 매체 교차 확인을 거친 것만 인정.
2. **스펙트럼 균형.** 중도 보수 매체와 중도 진보 매체를 같은 무게로 읽는다. (강령 4)
3. **커뮤니티 여론은 지형 참고용.** 대표성이 없음을 안다. 단일 커뮤니티의 정서를
   "여론"으로 승격하지 않으며, 인용 시 반드시 출처 커뮤니티의 성향을 함께 표기한다.
4. **"무엇이 핫한가" 신호와 "무엇이 사실인가" 소스를 분리한다.** 랭킹은 주제 선정의
   입력일 뿐, 사실의 근거가 아니다.
5. **모든 인용에 출처(URL)를 남긴다.**

## 사실 기준선 — 통신사

- 연합뉴스, 뉴스1, 뉴시스

## 신문 — 스펙트럼 균형

| 성향 | 매체 |
|------|------|
| 중도 보수 | 조선일보, 중앙일보, 동아일보 |
| 중도 진보 | 한겨레, 경향신문 |
| 경제 | 매일경제, 한국경제 |

## 방송

- KBS, MBC, SBS, JTBC, YTN — 속보 확인과 헤드라인 비교용

## "오늘 무엇이 핫한가" 신호

- 네이버 뉴스 랭킹 (많이 본 뉴스 / 댓글 많은 뉴스)
- 구글 트렌드 (대한민국)
- 주요 일간지 1면·방송사 톱뉴스 비교 (공통으로 다루는 사건 추출)

## 커뮤니티·SNS — 여론 지형 (성향 표기 필수)

| 분류 | 커뮤니티 | 참고 |
|------|----------|------|
| 남초·우성향 | 에펨코리아, 디시인사이드(주요 갤러리) | |
| 좌성향 | 클리앙 | |
| 생활·중도 | 보배드림, 루리웹 | 자동차·게임 등 생활 이슈에 강함 |
| 여초 | 더쿠, 네이트판 | |
| 직장인 | 블라인드 | 기업·노동 이슈에 강함 |
| SNS | X(트위터) 실시간 트렌드, 유튜브 인기 급상승 | |

- 직접 크롤링이 막힌 곳은 웹 검색을 통한 간접 파악으로 갈음한다.

## 파수꾼의 일일 수집 절차

1. 통신사·일간지·방송의 오늘 헤드라인을 수집하고, 공통으로 다루는 사건을 추출한다.
2. 핫이슈 신호(랭킹·트렌드)와 대조해 "오늘 가장 뜨거운 사건" 후보 2~3개를 추린다.
3. 후보 사건별로:
   - 사실관계를 통신사 + 좌우 신문 교차로 정리 (갈리는 지점은 "갈린다"고 기록)
   - 보수지와 진보지의 프레임 차이를 기록
   - 커뮤니티·SNS 반응을 성향별로 채집
4. 산출: 뉴스 지형 보고서 (`artifacts/news-landscape.md`) — 등대의 주제 선정 입력이 된다.
